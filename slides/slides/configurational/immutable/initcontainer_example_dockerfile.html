Example (Dockerfile)

  • Dockerfile for init container:
FROM busybox

ADD dev.properties /config-src/demo.properties
# ... add more to /config-src

# Using a shell here in order to resolve wildcards
ENTRYPOINT [ "sh", "-c", \
             "cp /config-src/* $1", "--" ]
  • Build config image:
docker build -t k8spatterns/config-dev:1 .
